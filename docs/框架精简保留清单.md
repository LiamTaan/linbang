# 邻里互助框架精简保留清单

## 1. 目标口径

本次 SQL 重整目标是：

- 从空库直接初始化邻里互助，不再要求先执行官方 `ruoyi-vue-pro.sql`
- 框架层只保留后台登录、鉴权、字典、日志、文件、短信、通知、支付、任务调度所需最小底座
- 业务层只保留当前邻里互助后端已实现、管理端已接入的模块
- 删除官方演示业务、无关菜单、无关字典、无关示例配置
